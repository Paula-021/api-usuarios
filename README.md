## Sobre o projeto

Esta API REST foi desenvolvida com Java e Spring Boot para gerenciar usuários e seus respectivos endereços, aplicando boas práticas de desenvolvimento back-end e arquitetura em camadas.

Durante o desenvolvimento, foi adotada a separação de responsabilidades entre Controllers, Services, Repositories, DTOs e Entidades, promovendo um código mais organizado, escalável e de fácil manutenção.

Além da implementação das operações de CRUD, o projeto contempla validação de dados de entrada, tratamento de exceções personalizadas e utilização do Spring Data JPA para persistência das informações.

### Principais conhecimentos aplicados

- Desenvolvimento de APIs REST com Spring Boot
- Arquitetura em camadas (Controller, Service e Repository)
- Modelagem de entidades com JPA/Hibernate
- Utilização de DTOs para transferência de dados
- Tratamento de exceções personalizadas
- Validação de dados de entrada
- Persistência de dados com Spring Data JPA
- Organização de código seguindo princípios de boas práticas

Este projeto foi desenvolvido com o objetivo de consolidar conhecimentos em desenvolvimento back-end utilizando o ecossistema Spring, simulando uma aplicação real e seguindo padrões amplamente utilizados no mercado.
